# Trace Survey User Interaction API Microservice

A public-facing microservice providing APIs for user interaction, built with Go.

## Overview

This microservice handles user interaction through RESTful APIs, providing a clean interface for client applications. It uses modern Go practices with a focus on performance, reliability, and maintainability.

## Features

- RESTful API endpoints for user interactions
- Authentication
- Request validation and error handling
- Containerized deployment with Docker
- CI/CD pipeline integration with conventional commits

## Prerequisites

- Go 1.21 or higher
- Docker
- Git
